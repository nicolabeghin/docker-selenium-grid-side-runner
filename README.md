# Dockerized Selenium HUB, node and side-runner all-in-one

Ready to use
* Selenium grid HUB
* Selenium Chrome node
* Selenium side-runner

## Quick start

* copy your `side`'s in `./sides` directory
* run `./test.sh`
* get result from console and `./out` directory (in json-format)
